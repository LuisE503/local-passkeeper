## Translation Guidelines

All translations follow these principles:

- **Consistency**: Same structure and sections as English version
- **Technical Accuracy**: Technical terms preserved or appropriately translated
- **Code Examples**: Preserved in original form with English comments
- **Links**: Updated to point to correct language versions
- **Completeness**: All sections translated, no machine-translation artifacts

## Adding New Translations

To add a new language:

1. Copy `../README.md` to `README.{language-code}.md`
2. Translate all content while maintaining structure
3. Update all cross-references to other translations
4. Keep code examples and technical terms consistent
5. Test all links
6. Submit a pull request
